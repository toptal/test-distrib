# distrib-core

Is a common core module for [rspec-distrib](../rspec-distrib).

## Installation

Add the gem to the application's Gemfile:

```ruby
gem 'distrib-core', git: 'git@github.com:toptal/distrib-core.git',
  glob: 'distrib-core/*.gemspec'
```

## Getting started

```shell
bundle install
bundle exec rspec
bundle exec rubocop
```
